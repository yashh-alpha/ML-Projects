# Machine Learning Projects

This repository contains a collection of machine learning projects focused on applying core concepts to real-world datasets. The projects are organized to demonstrate understanding of data preprocessing, model building, evaluation, and comparison across different algorithms.

## Overview

The purpose of this repository is to:

* Practice and implement machine learning techniques
* Compare different models on structured datasets
* Build intuition about model performance and behavior
* Maintain a structured portfolio of work

## Projects

Each project is contained in its own folder and typically includes:

* A Jupyter Notebook with implementation
* A dataset (if applicable)
* A project-specific README with details

## Topics Covered

* Regression and Classification
* Model Comparison
* Feature Scaling and Preprocessing
* Evaluation Metrics (R² Score, MSE, Accuracy, etc.)
* Basic Hyperparameter Tuning

## Tools and Libraries

* Python
* Pandas
* NumPy
* Scikit-learn
* XGBoost
* Matplotlib
